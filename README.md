# Tic-Tac-Toe AI Game

A simple Tic-Tac-Toe game where players can compete against an intelligent AI opponent. The AI adapts to your moves, providing a challenging experience for both casual players and those interested in game logic and AI strategies.

## Features
- Play against a smart AI opponent
- Clean and user-friendly interface
- Customizable gameplay options (e.g., starting player, board size)
- AI adapts to player strategies for more competitive gameplay

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/tictactoe-ai.git
   ```
2. Navigate to the project directory:
   ```bash
   cd tictactoe-ai
   ```
3. Run the game:
   - If it's a Python-based game:
     ```bash
     python tictactoe.py
     ```

## How to Play

1. Choose whether to play as X or O.
2. The game alternates turns between you and the AI.
3. Try to align three of your symbols horizontally, vertically, or diagonally to win!
